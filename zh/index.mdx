欢迎使用 **Aether** — 运行在 Android 上的本地 AI Agent。基于 Pi 框架构建，Aether 将完整的智能体体验直接带到你的设备：连接任意主流 LLM 提供商，借助内置 Alpine Linux 工具自动化任务，并通过自定义 mods 扩展 Agent — 全程无需将凭证经任何由 Aether 运营的服务中转。

## 快速上手

<Steps>
  <Step title="从 GitHub Releases 安装 Aether">
    从 [Aether Releases 页面](https://github.com/Zhou-Shilin/Aether/releases) 下载最新 APK，并安装到你的 Android 设备。
  </Step>
  <Step title="配置 LLM 提供商">
    打开 Aether，进入 **设置 → 模型提供商**，添加你偏好的 LLM 提供商，并配置 API 密钥、OAuth 或环境凭证。
  </Step>
  <Step title="开始与 AI Agent 对话">
    在顶栏选择模型并发送第一条消息。启用 **Agent 模式** 即可在隔离的 Android 虚拟显示上控制应用。
  </Step>
</Steps>

<Note>
  Aether 需要 **Android 8.0（API 26）或更高版本**，以及 **arm64-v8a** 设备架构。安装前请确认设备满足这些要求。
</Note>
