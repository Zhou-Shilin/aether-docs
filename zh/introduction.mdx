**Aether** 是基于 Pi 框架构建的 Android 通用 AI Agent。它将极简、精心打磨的界面与强大的智能体执行引擎相结合，开箱即用支持多种 LLM 提供商。无论你需要一个得力的 AI 聊天助手，还是一个能执行 shell 命令、控制手机的端侧自主 Agent，Aether 都能胜任。

## Aether 不是什么

Aether **不是** 云服务，也不是某一家 AI 提供商应用的套壳。你配置的所有 LLM API 密钥与凭证都保存在设备上的 Aether 私有应用目录中 — 不会经过任何中间服务器。你选择提供商，你拥有自己的数据。

## Pi 框架

Aether 构建于 **Pi agent 框架** 之上，将其作为核心执行引擎。Pi 提供：

- **广泛的 LLM 提供商兼容性** — 统一接口覆盖所有支持的提供商，切换模型无需改代码。
- **轻量级 Agent 执行** — 高效管理工具调用、上下文窗口与多步推理循环，开销可控。
- **可扩展钩子** — Aether 向开发者暴露的扩展点，直接内建于 Pi 架构中。

由于提供商通信与 Agent 编排委托给 Pi，Aether 保持精简的同时依然高度可用。

## 面向扩展开发者

Aether 的扩展系统允许你以自包含包的形式交付新工具、UI 元素与 Agent 行为。扩展可写成热重载 TypeScript mods 以便快速原型，也可写成编译型 Kotlin/DEX native mods，以便完整访问 Android API 并获得原生性能。
